## 📝 FEATURE 2: EXAM MODE

### How to Use
1. Click **📝 Exams** button
2. Enter:
   - 📖 **Subject** (e.g., Python, History, Biology)
   - 🎯 **Topic** (e.g., Lists, Ancient Rome, Cells)
   - ⚡ **Difficulty** (Easy / Medium / Hard)
3. Click **🚀 Generate Exam**
4. Answer 10 multiple-choice questions
5. Click **✅ Submit Exam**
6. Get instant score & XP reward!

### Difficulty Levels
- **Easy**: Basic concepts, definitions, simple applications
- **Medium**: Intermediate theory, typical applications
- **Hard**: Advanced concepts, complex scenarios, multi-step reasoning

### Scoring System
- 10 questions total
- You get: (correct answers / 10) × 100% score
- **XP Earned**: Score × 5 XP
  - 80% = 400 XP
  - 100% = 500 XP
  - 50% = 250 XP

### Performance Feedback
- 🌟 **90-100%**: "Excellent! You've mastered this topic!"
- 👍 **70-89%**: "Good effort! Review weak areas."
- 📚 **Below 70%**: "Keep practicing! You can do better."

### Pro Tips
- Start with Easy, then try Medium, then Hard
- Review weak areas between exams
- Take multiple exams on same topic
- See score improvement over time
- Each exam earns XP for leveling up!

### Example Workflow
```
Exam 1: Python Lists (Easy) → 80% → 400 XP
Exam 2: Python Lists (Medium) → 75% → 375 XP
Exam 3: Python Lists (Hard) → 85% → 425 XP
Total Learned: Advanced Python knowledge
XP Earned: 1200 XP → Level Up!
```
