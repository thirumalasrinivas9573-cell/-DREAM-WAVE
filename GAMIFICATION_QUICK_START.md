# Krishna AI Gamification - Quick Start Guide

## 🎮 What's New?

Your Krishna AI Mentor now has an **advanced gamification system** that turns personal growth into an addictive game!

---

## 📊 The Gamification Widget

Located at the top of the chat panel, showing:

```
    [⭐ 1]  🔥 Streak: 0  ⭐ Level: 1

    📊 Experience
    [████----------] 0/100 XP
```

### What Each Stat Means:

- **🔥 Streak**: Days of completed tasks (0-999)
- **⭐ Level**: Your progress rank (1-15+)
- **📊 XP Bar**: Progress to next level
- **🏆 Certificate**: Unlocks at Level 10

---

## 📋 Daily Tasks Tracker

Below the welcome card, you'll see 6 daily tasks:

```
📋 Today's Tasks (2/6)
  
  ☐ Morning Meditation (+10 XP)
    Description: 15 min guided meditation
  
  ☑ Read Sacred Text (+10 XP)  ✓
    Description: Read Bhagavad Gita or similar
  
  ☐ Mindful Exercise (+10 XP)
    Description: Yoga, Running, or Gym
  
  ... and 3 more tasks
```

### How to Use:

1. **Check off each task** as you complete it
2. **Earn +10 XP per task** (shown in gold)
3. **Complete all 6 tasks** to unlock the "Complete Day" button
4. **Click "Complete Day"** to earn +50 XP bonus + streak increase

## 🔥 Streak System

### How to Build a Streak:
1. ✅ Complete all 6 daily tasks every day
2. 🎉 Get +1 to your streak
3. 🏆 Continue the next day to grow it!

### What Breaks a Streak?
- ❌ Miss 1 day → Warning (lose 1 streak)
- ❌ Miss 2+ days → Streak resets to 0
- ⚠️ You'll see a warning message to save your streak

### Example Streak Growth:
```
Day 1: Complete all tasks → Streak = 1 ✓
Day 2: Complete all tasks → Streak = 2 ✓
Day 3: Miss tasks → Streak = 1 ⚠️ (warning)

Day 5: Complete all tasks → Streak = 3 ✓
Day 6: Complete all tasks → Streak = 4 ✓
Day 7: Complete all tasks → Streak = 5 ✓
Day 7: Level Up! 🎉
```

## ❓ FAQ

**Q: What if I forget a task?**
A: No XP for that task, and no streak bonus. The day still completes if you do finish other tasks.

**Q: Can I set custom tasks?**
A: Currently 6 preset tasks. Future version will allow customization!

**Q: What if I uninstall the app?**
A: Your data stays in browser storage. It won't sync to another device.

**Q: Can I reset my progress?**
A: Clear browser localStorage for the domain to reset (browser console: `localStorage.clear()`)

**Q: How do I get to Level 100?**
A: ~10,000+ XP needed! That's about 95+ days of full completion.
